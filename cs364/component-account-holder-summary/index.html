Account Holder Summary

Normal View:

User Name Account Balance Interest Rate

Example Normal View:

Jennifer $153.50 50% APR

Simple View:

User Name $Balance

Example Simple View:

Jennifer $153.50

Endpoint Calls:

  • account-holder/{accountHolderId}/summary/
    • Description: the account holder summary is displayed many places
    • Method: Get
    • Parameters Passed: (accountHolderUserId)
    • Expected Return:
      • Success: JSON object with username, accountBalance, interestRate
      • Failure: {error message}
  • account-holder/{accountHolderId}/summary/{category}
    • Description: the account holder category summary when user drills into a specific category
    • Method: Get
    • Parameters Passed: (accountHolderUserId, category)
    • Expected Return:
      • Success: JSON object with username, categoryBalance
      • Failure: {error message}
