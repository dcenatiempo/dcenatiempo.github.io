Category Settings

Category: Goal: Recurring Transactions:
Transactions List Component Filtered by Recurring Transactions in this category

Endpoints:

  • /category/settings/{accountHolderId}/{category}
    • Description: returns settings for a specific category for a specific account holder
    • Method: Get
    • Parameters Passed: (accountHolderId, category)
    • Expected Return:
      • JSON object category info: {category, balance, goalAmount, [array of recurring transactions for this category]
      • Failure: {error message}
  • /category/settings/{accountHolderId}/{category}
    • Description: makes changes to category settings for a specific category of a specific acccount holder
    • Method: Get
    • Parameters Passed: (accountHolderId, category, object with changes})
    • Expected Return:
      • Success: {success message}
      • Failure: {error message}
