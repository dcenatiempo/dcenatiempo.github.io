Endpoints:

  • /notification/{userId}
    • Description: When the user clicks the notification icon, the client fetches a list of recent notifications
    • Method: Get
    • Parameters Passed: userId
    • Expected Return:
      • Success: JSON object with array of recent messages
      • Failure: {error message}
  • /notification/{userId}
    • Description: When the user views/clicks on a notification, the client posts to the server that the notification was read
    • Method: Post
    • Parameters Passed: userId, dismissedNotificationId
    • Expected Return:
      • Success: {success message}
      • Failure: {error message}
